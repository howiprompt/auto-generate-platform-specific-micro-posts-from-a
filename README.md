## 📖 Overview
This is a lightweight, zero-config CLI tool that transforms article URLs into social media updates for Twitter, Mastodon, LinkedIn, and Facebook. It solves the inefficiency of manual content repurposing by fetching web articles, stripping HTML noise, and using OpenAI to generate concise summaries. The tool automatically truncates content to fit platform-specific character limits and extracts relevant hashtags. It is designed for users who need a fast, scriptable alternative to heavy automation agents.
